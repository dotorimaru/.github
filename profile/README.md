#### 커밋메시지 컨벤션 (commit message convention)
```
- feat: 새로운 기능 추가

- fix: 버그 수정

- build: 빌드 관련 파일 수정

- chore: 코드의 변경 없이 설정을 변경

- refactor: 코드 리팩토링(재설계)

- rename: 파일명 또는 폴더명 수정

- remove: 필요 없는 코드 또는 파일 삭제

- docs: 문서(Readme) 수정
```
